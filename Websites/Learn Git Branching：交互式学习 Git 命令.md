# 简介

“Learning Git Branching” 可以说是目前为止学习 Git 最好的教程了，在沙盒里你能执行相应的命令，还能看到每个命令的执行情况；通过一系列刺激的关卡挑战，逐步深入的学习 Git 的强大功能，在这个过程中你可能还会发现一些有意思的事情。
